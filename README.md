# 🌱 Plantory – Full-Stack Plant Inventory System

Plantory is a modern full-stack CRUD web application built with **Next.js** that allows users to manage a dynamic plant catalog. It features secure authentication via **GitHub OAuth**, persistent data storage with **PostgreSQL (Neon)**, and powerful database interaction using **Prisma ORM**. The UI is fully responsive and includes **dark/light mode** using **ShadCN UI components**.

## 🚀 Features

- 🔐 GitHub OAuth authentication using NextAuth.js
- 🪴 Full CRUD functionality (Create, Read, Update, Delete) for plant entries
- 🌗 Dark/Light mode toggle with ShadCN UI
- 🧠 Database interaction with Prisma ORM
- ☁️ PostgreSQL hosted on Neon
- 📱 Responsive design with Tailwind CSS

## 🛠 Tech Stack

| Tech              | Description                               |
